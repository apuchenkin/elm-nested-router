# Elm nested router

A simple router for single page applications, written in Elm



Example: [live demo](http://apuchenkin.github.io/elm-nested-router/example)

### Currently supports
- [x] HTML5 push state
- [x] Route params
- [x] Params constraints (String, Int, Enum, Regexp)
- [x] Optional params
- [x] Named views
